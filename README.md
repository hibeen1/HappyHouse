# Happy House
### 💡 프로젝트 개요
- "Happy House"는 공공데이터와 카카오 지도 API를 활용한 아파트 실거래 목록 조회 서비스 입니다.
- 부동산 시장 관심이 많은 요즘 아파트 실거래 정보를 사용자에게 쉽고 편리하게 제공하기 위해 프로젝트를 진행하였습니다.
